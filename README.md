# quickplay
small script to play to download a youtube video, play with mpv and then delete
## requirements
- youtube-dl
- mpv
## instructions
- clone this repository
`git clone https://github.com/realmonado/quickplay.git`
- enter directory
`cd quickplay`
- make the scipt executable
`chmod +x quickplay.sh`
- execute the script
`./quickplay.sh`
